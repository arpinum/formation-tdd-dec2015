# Kata Mölkky en Java

Implémentation du calcul du score au Mölkky.

Les règles sont disponibles sur [wikipedia].

## Run unit tests

    ./gradlew test

(use .bat on windows)

## Run lint, unit tests and coverage

    ./gradlew

(use .bat on windows)

## Bibliothèques utilisées

* Assertion api: [AssertJ]
* Test runner: [JUnit]
* Linter: [PMD]
* Coverage: [JaCoCo]
* Mock api: [Mockito]
