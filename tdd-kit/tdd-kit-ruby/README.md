# TDD Kit - Ruby [![Travis status]](https://travis-ci.org/arpinum/tdd-kit-ruby)

## Initialization

    rvm install 2.2.3
    bundle install

## Run unit tests

    rake test

## Run lint, unit tests and coverage

    rake

## Libraries documentation

* Assertion api: [Minitest expectations]
* Linter: [Rubocop]
* Cover: [SimpleCov]
* Mock api: [Mocha]
